# 零代码工具平台详细规划

## 项目概述

**项目名称**: DevToolsKit (或 ZeroCodeKit)**技术栈**:

- Vue 3 + TypeScript + Vite
- Vue Router (路由管理)
- Pinia (状态管理)
- Element Plus / Ant Design Vue (UI组件库)
- Monaco Editor (代码编辑器)
- CodeMirror 6 (轻量编辑器)
- Mermaid.js (图表渲染)
- marked + highlight.js (Markdown渲染)
- 其他工具库见依赖规划

## 核心功能模块详细设计

### 1. API 测试/文档生成模块

**功能特性**:

- HTTP请求构建器（GET/POST/PUT/DELETE等）
- 请求头编辑器（支持自定义headers）
- 请求体编辑器（JSON/Form-data/Raw）
- 响应查看器（格式化JSON、状态码、响应时间）
- 请求历史记录（本地存储）
- 自动生成API文档（Markdown格式）
- 支持环境变量（baseURL等）
- CORS代理提示（纯前端限制）

**技术实现**:

- 使用 `axios` 或原生 `fetch` 发送请求
- 使用 `Monaco Editor` 编辑JSON
- 使用 `marked` 生成文档
- 本地存储使用 `localStorage`

**UI布局**:

```javascript
┌─────────────────────────────────────────┐
│  [Method] [URL输入框] [Send] [Save]     │
├─────────────────────────────────────────┤
│  Headers | Body | Params | Docs         │
│  ┌───────────────────────────────────┐  │
│  │  Key: Value 编辑器               │  │
│  └───────────────────────────────────┘  │
├─────────────────────────────────────────┤
│  Response (200 OK)                      │
│  ┌───────────────────────────────────┐  │
│  │  格式化JSON响应                   │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```



### 2. 数据库查询构建器模块

**功能特性**:

- 可视化表选择器
- 字段选择（支持多选）
- WHERE条件构建器（拖拽式）
- JOIN关系构建器
- GROUP BY / ORDER BY 配置
- 实时SQL预览
- SQL格式化
- 支持MySQL/PostgreSQL/SQLite语法

**技术实现**:

- 使用拖拽库 `vue-draggable` 或 `@dnd-kit`
- SQL AST构建和生成
- 使用 `sql-formatter` 格式化SQL

**UI布局**:

```javascript
┌─────────────────────────────────────────┐
│  Tables: [选择表]                       │
├─────────────────────────────────────────┤
│  Fields: [多选字段]                     │

│  ┌───────────────────────────────────┐  │

├─────────────────────────────────────────┤
│  Generated SQL:                         │

│  │  SELECT ... FROM ... WHERE ...    │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```



### 3. 表单构建器+数据收集模块

**功能特性**:

- 拖拽式表单字段添加
- 字段类型：文本、数字、邮箱、日期、选择、多选、文件上传
- 字段配置面板（标签、验证规则、必填等）
- 实时预览表单
- 表单数据收集（JSON格式）
- 数据导出（CSV/JSON）
- 表单JSON导出/导入

**技术实现**:

- 使用 `vue-draggable` 实现拖拽
- 动态表单渲染
- 表单验证使用 `vee-validate` 或自定义

**UI布局**:


**功能特性**:

- 分屏编辑（左侧编辑，右侧预览）

**卡片设计**:

- 阴影: `0 4px 20px rgba(0, 0, 0, 0.4)` (深色阴影)
- 悬浮效果: `transform: translateY(-2px); box-shadow: 0 8px 30px rgba(102, 126, 234, 0.3);`
- 边框光晕: `box-shadow: 0 0 20px rgba(102, 126, 234, 0.1);`

**按钮设计**:

- 主按钮: 渐变背景 + 光晕效果
- 悬浮: 放大1.05倍 + 增强光晕
- 点击: 缩小0.98倍
- 禁用: 降低透明度到0.5

**输入框设计**:

- 背景: 半透明深色
- 边框: 默认半透明，聚焦时发光
- 聚焦光晕: `box-shadow: 0 0 0 3px rgba(102, 126, 234, 0.2);`
